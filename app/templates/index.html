AI Ethical Decision-Making Simulator

A simulation application for event-based scenarios with temporal reasoning, focusing on military medical triage as the initial use case.

This application simulates events with temporal features, allowing for the evaluation of decisions through both a rules-based engine (for constraint validation) and a case-based/guideline-based system (for ethical assessment).

Explore Scenarios
Event Representation

Utilizes LangGraph for orchestrating complex decision-making workflows with built-in state management.

Knowledge Base

Integrates domain-specific knowledge and ethical guidelines using LangChain's document loaders, retrievers, and vector stores. Leverages Model Context Protocol (MCP) for accessing domain-specific guidelines and case repositories.

Decision-Making Framework

Implements a hybrid ethical reasoning system combining rule-based, case-based, and principle-based reasoning.

Getting Started

To begin using the simulator:

  1. Explore existing scenarios or create a new one
  2. Add characters, conditions, and resources to your scenario
  3. Create a timeline of events
  4. Make decisions and see how they are evaluated

Use Cases

The initial use case focuses on military medical triage scenarios, where medical personnel must make rapid decisions about patient treatment priorities under resource constraints and ethical considerations.

Future use cases may include:

  • Disaster response and humanitarian aid
  • Resource allocation in healthcare systems
  • Ethical decision-making in autonomous systems
